[MyBatis SQL Log Merger] 隐私权政策

生效日期：2025年12月15日

引言

我们重视您的隐私。为[MyBatis SQL Log Merger]（以下简称“本插件”）的用户提供此隐私权政策，详细说明我们如何收集、使用、存储及保护您的信息。

一、数据收集与使用

1. 收集的数据类型

  • 设备与技术信息：浏览器版本、操作系统、IP地址（仅用于优化服务，不关联个人身份）。
  • 使用数据：卡片布局偏好、高频访问网站、插件设置选项。
  • 用户创建内容：您添加到导航卡片的网址（仅存储加密哈希值，不记录完整URL）。

2. 数据用途

我们仅将数据用于：

  • 提供并优化导航页功能
  • 生成个性化推荐（需开启「智能推荐」功能）
  • 保障服务安全与稳定性

二、数据共享

除非法律要求，我们不会与任何第三方共享数据。例外情况包括：

  • 云同步服务：当您登录账号启用跨设备同步时，数据会加密传输至我们的服务器（详见数据存储章节）
  • 服务提供商：仅限托管、支付处理等必要合作方（需符合GDPR等合规要求）

三、数据存储

1. 本地存储

默认情况下，所有卡片数据、配置信息均存储在您的本地浏览器中，我们无法访问。

2. 云端存储（可选）

若您选择同步数据至云端：

  • 数据采用AES-256端到端加密
  • 服务器仅保留数据30天（到期自动删除）
  • 严格限制员工访问权限

四、用户权利

您有权：

  • 随时通过插件设置查看/删除数据
  • 撤回同步授权
  • 要求导出您的数据（CSV格式）

五、数据保留

我们仅在以下期限内保留数据：

  • 账户活跃期 + 90天
  • 法律要求的最长期限（如适用）

六、安全措施

  • 传输层使用TLS 1.3加密
  • 定期进行第三方安全审计
  • 自动清除7天未操作的临时缓存

七、儿童隐私

我们不收集13岁以下儿童的个人信息。若发现误收数据，将立即删除。

八、政策更新

重大变更将通过插件内通知及邮件告知，更新版本于公布后30天生效。
