  • 将您的MyBatis日志粘贴到下方的输入框中
  • 点击"处理SQL"按钮将日志转换为完整的SQL语句
  • 转换后的SQL将显示在结果区域
  • 使用"清空"按钮可以清除所有内容
  • 使用"美化SQL"按钮可以格式化并高亮显示SQL语句
  • 使用"拷贝结果"按钮可以将SQL复制到剪贴板
  • 可以切换输入模式：单框模式（粘贴完整日志）或双框模式（分别输入SQL模板和参数）
  • 双框模式支持带或不带Preparing:和Parameters:前缀的输入

输入日志片段：

==>  Preparing: SELECT * FROM users WHERE id = ? AND name = ?
==> Parameters: 123(Long), "John"(String)

输出SQL：

SELECT * FROM users WHERE id = 123 AND name = 'John'

双框模式输入示例：

SQL模板（可带或不带Preparing:前缀）：

Preparing: SELECT * FROM users WHERE id = ? AND name = ?
或
SELECT * FROM users WHERE id = ? AND name = ?

参数（可带或不带Parameters:前缀）：

Parameters: 123(Long), "John"(String)
或
123(Long), "John"(String)
